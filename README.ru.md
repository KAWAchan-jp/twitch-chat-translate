# Twitch Chat Translator

[English](README.en.md) | [日本語](README.md) | Русский

Веб-приложение для перевода и отображения чата Twitch в реальном времени.

**[▶ Использовать сейчас](https://kawachan-jp.github.io/twitch-chat-translate/)**

---

## Возможности

- **API-ключ не нужен** - Используется неофициальный API Google Translate, поэтому можно начать без регистрации
- **Сервер не нужен** - Приложение полностью работает в браузере и запускается на GitHub Pages
- **Анонимное подключение** - Можно читать чат Twitch без аккаунта Twitch
- **Автоопределение языка** - Исходный язык определяется автоматически
- **Поддержка нескольких языков** - Поддерживается более 15 языков
- **Распознавание эмотов** - Повторяющиеся эмоты автоматически сворачиваются
- **Настройка шрифта и размера** - Можно настроить отображение чата для удобного чтения
- **Оверлей OBS** - Отображение оформленного чата как браузерного источника OBS
- **Настройка оформления** - Пресеты, цвета, интервалы, время отображения и эффекты
- **Пользовательский CSS** - Точная настройка отображения чата с помощью CSS
- **Отправка сообщений (экспериментально)** - Войдите через Twitch и отправляйте переведенные сообщения в чат

## Как использовать

### 1. Подключитесь к каналу

![Экран настройки](docs/images/screenshot-setup.png)

1. [Откройте страницу](https://kawachan-jp.github.io/twitch-chat-translate/)
2. Введите имя канала, например `xqc`
3. Выберите **исходный язык**. Если вы не уверены, оставьте "Автоопределение"
4. Выберите **язык перевода**. По умолчанию используется японский
5. Нажмите **Подключиться**

### 2. Используйте экран чата

![Экран чата](docs/images/screenshot-chat.png)

После подключения настройки можно менять в верхней панели.

| Элемент | Описание |
|---------|----------|
| `Авто → Японский` | Переключение исходного языка и языка перевода в реальном времени |
| `S / M / L / XL` | Изменение размера текста чата |
| `⚙` | Открыть панель расширенных настроек |
| `Отключиться` | Отключиться от канала и вернуться на начальный экран |

### 3. Панель настроек

Нажмите **значок ⚙** в верхней панели, чтобы открыть настройки.

| Настройка | Описание |
|-----------|----------|
| Автопрокрутка | Автоматически прокручивать вниз при появлении новых сообщений |
| Скрывать ботов и команды | Исключать сообщения ботов и команды из отображения чата |
| Отправка [экспериментально] | Показать панель отправки сообщений. Требуется вход через Twitch |
| Шрифт | Переключение между гротеском, антиквой и моноширинным шрифтом |
| Язык интерфейса | Изменить язык интерфейса |
| Оформление | Изменить пресеты, цвета, интервалы и эффекты чата |
| Управление заблокированными пользователями | Добавить или удалить скрытых пользователей |
| Копировать URL для OBS | Скопировать URL браузерного источника OBS с текущими настройками оформления |

## Оформление

Нажмите **Оформление** в панели настроек, чтобы настроить отображение чата.

| Элемент | Описание |
|---------|----------|
| Пресет | Plain / Box / Separator / Bubble / Glass / Neon / Compact / Terminal / Minimal / Card / Broadcast / Pastel |
| Показывать элементы | Показать или скрыть имя, время, оригинал и перевод |
| Эффект | Выбрать анимацию появления нового сообщения |
| Цвет фона | Задать общий фон сообщений для всех пресетов |
| Цвета текста | Отдельно изменить цвета перевода, оригинала и имени |
| Восстановить цвета пресета | Вернуть стандартные цвета выбранного пресета |
| Интервал между сообщениями | Настроить расстояние между сообщениями |
| Исчезновение по времени | Удалять сообщения через заданное время с эффектом Fade / Slide / Shrink / Blur |
| Цвет хромакея | Задать цвет фона, удаляемый хромакеем OBS |
| Пользовательский CSS | Напрямую редактировать CSS для точной настройки |

Настройки автоматически сохраняются в браузере. Пользовательский CSS также сохраняется и применяется во время ввода, поэтому кнопка сохранения не требуется.

### Отображение в OBS

1. Настройте отображение чата в разделе **Оформление**
2. Нажмите **Копировать URL для OBS** в панели настроек
3. Добавьте браузерный источник в OBS и вставьте скопированный URL
4. При необходимости настройте хромакей в OBS

Скопированный URL для OBS содержит настройки оформления, отображаемые элементы, цвета, интервалы, время отображения и пользовательский CSS.

## Отправка сообщений (экспериментально)

![Панель отправки](docs/images/screenshot-send.png)

После входа через аккаунт Twitch вы можете переводить и отправлять сообщения в чат.

1. Включите **Отправка [экспериментально]** в панели настроек
2. Нажмите **Войти через Twitch** и авторизуйте приложение
3. Введите сообщение, затем нажмите `Enter` или **Отправить**
4. Введенный текст автоматически переводится на **язык чата** и отправляется

> **Примечание:** Если исходный язык установлен на "Автоопределение", сообщение будет отправлено без перевода. Чтобы отправить перевод, выберите исходный язык вручную.

## Использование на своем сервере

Вы также можете скачать этот репозиторий и разместить его на своем сервере или собственном домене.

Если вам нужно только **читать и переводить чат**, обычно достаточно загрузить все файлы на веб-сервер. API-ключ и настройка интеграции с аккаунтом Twitch не требуются.

Однако, если вы хотите **входить через Twitch и отправлять сообщения в чат**, нужно зарегистрировать свое приложение в Twitch Developer Console.

1. Откройте [Twitch Developer Console](https://dev.twitch.tv/console)
2. Войдите через свой аккаунт Twitch
3. Зарегистрируйте приложение
4. Добавьте URL вашего сайта как `OAuth Redirect URL`
   - Пример: `https://example.com/`
   - Для локального тестирования: `http://localhost:8000/`
5. Укажите выданный `Client ID` и URL перенаправления в `js/config.js`

```js
export const TWITCH_CLIENT_ID = 'ваш Client ID';
export const TWITCH_REDIRECT_URI = 'https://example.com/';
```

Итого: для чтения и перевода чата достаточно разместить файлы на сервере. Настройка Twitch Developer Console нужна только для отправки сообщений в чат.

## Распознавание штампов / эмотов

Сообщения, где одно и то же слово повторяется три или более раза подряд, считаются спамом эмотами и сворачиваются без перевода.

```text
🎴 Штамп  karubisUnun  ×4
```

## Требования

- Работает в основных браузерах на ПК и смартфонах
- Рекомендуются Google Chrome / Microsoft Edge

---

## Технологии

- Twitch IRC over WebSocket (анонимное подключение / OAuth-аутентификация)
- Неофициальный API Google Translate (автоопределение языка через `sl=auto`)
- Twitch Helix API (получение имени вошедшего пользователя)
- Чистые HTML + CSS + JavaScript (без фреймворков)

---

## Примечания

- Неофициальный API Google Translate может измениться или перестать работать без предупреждения
- На каналах с очень активным чатом перевод может не успевать за сообщениями
- Разработчик не несет ответственности за любой ущерб, вызванный использованием этого инструмента
